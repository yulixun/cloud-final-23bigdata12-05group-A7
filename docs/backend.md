# 后端服务接口文档（Windows WSL2环境）
## 1. 服务信息
- 框架：Flask 2.3.3
- 运行环境：Windows WSL2 + Docker
- 监听端口：5000（宿主机映射）
- 依赖服务：MySQL 8.0（需先启动并健康检查）

## 2. 核心接口

- 路径：`/`
- 方法：GET
- 请求参数：无

  ```json
